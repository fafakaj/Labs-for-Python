# Лабораторная работа: Оптимизация численного интегрирования

## Описание

В ходе лабораторной работы были реализованы и протестированы 5 итераций оптимизации функции численного интегрирования:

- **Итерация 1**: Обычная Python-функция.
- **Итерация 2**: Оптимизация с помощью **потоков** (`ThreadPoolExecutor`) — **без ускорения** из-за GIL.
- **Итерация 3**: Оптимизация с помощью **процессов** (`ProcessPoolExecutor`) — **реальный параллелизм**, но с накладными расходами.
- **Итерация 4**: Оптимизация с помощью **Cython** — **значительное ускорение** за счёт компиляции в C.
- **Итерация 5**: Использование **`noGIL`** в Cython — **наилучший результат** с **потоками** (реальный параллелизм без накладных расходов на процессы).

## Вывод

`noGIL` + Cython + потоки показали **наилучшую производительность**.

## Технологии

 Cython, `ThreadPoolExecutor`, `ProcessPoolExecutor`, `noGIL`.
